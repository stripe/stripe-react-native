# react-native-stripe-sdk

Stripe SDK for React Native

## Installation

```sh
npm install react-native-stripe-sdk
```

## Usage

```js
import StripeSdk from "react-native-stripe-sdk";

// ...

const result = await StripeSdk.multiply(3, 7);
```
